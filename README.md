# ❌⭕ Tic-Tac-Toe (XO) Game

A classic, fully functional **Tic-Tac-Toe (XO)** desktop game developed using **C#** and **Windows Forms (.NET)**. This project focuses on graphical UI event handling, state management, and dynamic win-logic evaluation.

## 🚀 Features
* **Interactive Grid:** A responsive 3x3 game board utilizing graphical buttons.
* **Turn Management:** Seamless turn switching between Player X and Player O.
* **Win/Draw Detection:** Built-in algorithm that instantly checks rows, columns, and diagonals for a winner, or declares a draw if the board is full.
* **Score Tracker / Status:** Displays the current game status and which player's turn it is.
* **Reset Functionality:** A restart button to clear the board and start a new match instantly.

## 🛠️ Tech Stack
* **Language:** C#
* **Framework:** .NET / Windows Forms (WinForms)
* **IDE:** Visual Studio
